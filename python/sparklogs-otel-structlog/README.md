# SparkLogs Python + structlog + OpenTelemetry

Runnable project for [Python ingestion](https://sparklogs.com/docs/ingest/data-sources/languages/python) (structlog). See also the [OpenTelemetry SDKs guide](https://sparklogs.com/docs/ingest/data-sources/languages/opentelemetry-sdks).

On Windows, use **WSL** for `make mock-test`.

```bash
make mock-test
```
