# SparkLogs Ruby + OpenTelemetry logs (beta)

Runnable project for [Ruby ingestion](https://sparklogs.com/docs/ingest/data-sources/languages/ruby). See also the [OpenTelemetry SDKs guide](https://sparklogs.com/docs/ingest/data-sources/languages/opentelemetry-sdks).

**Stability:** OpenTelemetry Ruby logs are still evolving; gems are pinned in the `Gemfile`.

On Windows, use **WSL** for `make mock-test`.

```bash
make mock-test
```
