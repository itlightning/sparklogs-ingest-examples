# SparkLogs Go OTel example — logrus

Runnable project for the [Go ingestion guide](https://sparklogs.com/docs/ingest/data-sources/languages/go) (logrus option). See also the [OpenTelemetry SDKs guide](https://sparklogs.com/docs/ingest/data-sources/languages/opentelemetry-sdks).

**Stability:** Go OTel logs APIs are pre-1.0; pin versions together when upgrading.

On Windows, run `make mock-test` inside **WSL**.

```bash
make mock-test
```
